# Screenshot Images

Add the following screenshots from the CoPilot app:

1. `screen-home.png` - Home screen (portfolio overview)
2. `screen-discover.png` - Discover/Leaderboard screen
3. `screen-creator.png` - Creator profile screen
4. `screen-activity.png` - Trade activity screen
5. `og-image.png` - Open Graph image for social sharing (1200x630px)

## How to capture screenshots

1. Open the app at `http://localhost:5173` (or your dev server)
2. Use browser DevTools to set viewport to 390x844 (iPhone 14 Pro)
3. Take screenshots of each screen
4. Save with the filenames above

## Recommended dimensions

- Phone screenshots: 390x844px (or similar iPhone aspect ratio)
- OG image: 1200x630px
